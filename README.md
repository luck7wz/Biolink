🧬 BioLink

O BioLink é uma aplicação web desenvolvida em Python utilizando o framework Flask, voltada para o gerenciamento de usuários e agendamentos. O sistema permite organizar atendimentos e armazenar informações de forma simples e eficiente.

🚀 Funcionalidades
Cadastro de usuários
Listagem de usuários
Edição e exclusão de usuários
Agendamento de serviços
Visualização de agendamentos
Atualização e remoção de agendamentos
Validação de dados nos formulários
🛠️ Tecnologias Utilizadas

Python, Flask, Flask-SQLAlchemy, SQLite, HTML5 e CSS3.

⚙️ Como Executar o Projeto
Clone o repositório:
git clone https://github.com/SEU-USUARIO/BioLink.git
Acesse a pasta:
cd BioLink
Instale as dependências:
pip install flask flask_sqlalchemy
Execute o projeto:
python app.py
Acesse no navegador:
http://127.0.0.1:5000
📌 Objetivo do Projeto

Este projeto foi desenvolvido com fins educacionais, com o objetivo de praticar conceitos de desenvolvimento web, integração com banco de dados e criação de sistemas completos utilizando Python e Flask.
